**CSS(Cascading Style Sheets)** is used to format and set the style of HTML elements. 

CSS is used to define the style of each class or type of HTML as shown below:

```css
body {
  background-color: black;
}

h1 {
  color: white;
  text-align: center;
}

p {
  font-family: helvetica;
  font-size: 10px;
}
```
